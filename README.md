# Health Economics Unit — Data Visualisation Standards

This repository defines the Health Economics Unit’s standards for producing clear, consistent and accessible data visualisations across our analytical work.  
It includes written guidance and ready-to-use templates for **Python (matplotlib)** and **R (ggplot2)**.

## What’s in this repo

- **Standards**: principles, chart guidance, uncertainty, accessibility, and do/don’t examples
- **Templates**:
  - Python: a matplotlib style file and helper functions
  - R: a ggplot2 theme and example gallery
- **Governance**: publishing checklist to support safe and consistent sharing
